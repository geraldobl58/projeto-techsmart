# TechSmart - Gestão de Automação Industrial

Este projeto foi desenvolvido para o desafio de **Personalização e Integração Web** da UniFECAF. Trata-se de uma solução voltada para pequenos negócios que necessitam de uma presença online profissional para triagem de serviços técnicos.

## 🚀 Funcionalidades
- **Landing Page Responsiva:** Design industrial focado em conversão.
- **Validação com JavaScript:** O formulário impede envios com dados incompletos, alterando visualmente as bordas (feedback em tempo real).
- **Customização CSS:** Uso de animações Bézier para interatividade nos cartões de serviços.

## 🛠️ Tecnologias Utilizadas
- **HTML5 Semântico:** Uso de tags estruturais para acessibilidade.
- **CSS3 (Web Standards):** Flexbox, Grid e Media Queries.
- **JavaScript Vanilha:** Lógica de validação sem dependências externas.
- **Hospedagem:** GitHub Pages (Custo zero e alta performance).

## 📄 Como usar
1. Navegue pela seção de serviços para visualizar os efeitos de escala.
2. No formulário de contato, insira o telefone. O botão de envio só será habilitado após a inserção de pelo menos 10 caracteres.
3. Clique em enviar para processar o chamado técnico.
